Cloud Programming Portfolio — DLBSEPCP01_D — IU Internationale Hochschule

Diese Seite wird über Azure App Service (Free F1) gehostet und über Azure CDN global verteilt.

Cloud-Architektur

Die Infrastruktur basiert auf einer mehrstufigen Architektur mit folgenden Komponenten:

KomponenteAzure ServiceFunktion
WebhostingApp Service (F1/S1)Hosting der statischen Website
CDNAzure CDNGlobale Inhaltsverteilung via Edge Server
MonitoringApplication InsightsTelemetrie und Performance-Tracking
AlertingAzure MonitorMetriken, Alerts und Dashboards
SkalierungAuto-Scale (S1 Tier)CPU-basiertes horizontales Scaling

Infrastructure as Code

Die gesamte Infrastruktur wird über ein parametrisiertes ARM Template (azuredeploy.json) definiert und ist vollständig reproduzierbar. Der Upgrade-Pfad von Free F1 auf Standard S1 ist über einen einzigen Parameter (appServicePlanSku) steuerbar.

Sicherheitsmerkmale

HTTPS-Only Enforcement, TLS 1.2 Minimum, FTPS deaktiviert, CDN mit ausschließlich HTTPS-Zugriff.

Tier: Free F1 (Proof of Concept) — Upgrade-Pfad zu Standard S1 parametrisiert

Region: West Europe
